利用信号灯P-V操作实现同步

  • 同步机制实质

    1. 运行条件不满足时，能让进程暂停；
    2. 运行条件满足时，能立即让进程继续。

  • 关键操作

    例如：司机的起步，停车和售票员的关门，开门。
    关键操作是指会对其它进程造成影响或者要满足某些条件才能运行的操作。

  • 司机VS售票员

    int S1=0;门是否关好？0：没有；1：关好。
    int S2=0;车是否挺稳？0：没有；1：挺稳。

    司机进程
while(true)
{
 P(S1);
 起步；
 行驶；
 停车；
 V(S2);
}

    售票员进程
while(true)
{
 关门；
 V(S1);
 售票;
 P(S2);
 开门；
}
